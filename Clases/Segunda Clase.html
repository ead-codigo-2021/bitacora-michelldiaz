HTML : hyper text markup language

Esta clase fue una revisión sobre HTML.

Recomendaciones para escribir en Código:

  • utilizar minúsculas
  • utilizar letras, números y guiones
  • utilizar guiones para separar palabras
  • definir nombres cortos y ordenados

Estructura Base Al escribir “html” VS Code nos dará una ayuda de código, esta es la estructura base la escritura de código en HTML Semántica Utilizar las etiquetas correctas para cada caso de escritura. div: en línea span: en bloque

Imágenes: Para poner imágenes se debe utilizar la abreviación “img” (ejemplo): En “src” pondremos el nombre del archivo tal cual y en “alt” un comentario sobre la imagen
