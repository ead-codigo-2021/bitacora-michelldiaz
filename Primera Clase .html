Anidación

Una etiqueta puede contener a otra etiqueta

Indentación

Aplicar sucesivos niveles de márgenes de acuerdo a la anidación de etiquetas. Permite identificar con mayor facilidad problemas de anidación o desbalance en etiquetas de cierre. Se puede indentar con tabuladores o espacios

Recomendaciones de escritura en HTML (nombres de archivo y carpetas)

Preferir minúsculas

Utilizar sólo letras, números y guiones. Evita usar caracteres inusuales, tildes, símbolos, comillas, etc

Utilizar guiones para separar palabras. Evita siempre usar espacios

Definir nombres cortos y relevantes, junto con una convención de nombres
